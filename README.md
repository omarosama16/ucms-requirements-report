# 📄 UCMS Requirements Elicitation Report

This repository contains the **Requirements Elicitation Report** for the University Course Management System (UCMS).  
It focuses on gathering and analyzing user needs from students, faculty members, and parents to define system requirements.

---

## 🎯 Project Info
- **Project:** University Course Management System (UCMS)  
- **Date:** 21-04-2025  
- **Phase:** Requirements Analysis  

---

## 👥 Stakeholders
- Students  
- Faculty Members  
- Parents  

---

## 🧠 Elicitation Method
- Online Survey (Google Forms)

---

## 📊 Key Insights

- **Students:** Want GPA tracking, notifications, and easy access to materials  
- **Faculty:** Need grading tools, analytics, and student tracking  
- **Parents:** Focus on weekly reports and performance monitoring  

---

## 📌 Key Outcome
The report defines the core functional and non-functional requirements that guide the UCMS design and development phases.
